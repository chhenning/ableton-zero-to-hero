# Learning Path

## 1. TAETRO - Ableton Live For Beginners

Nice course that seems to touch everything from UI, MIDI controller, loops, etc

[playlist](https://www.youtube.com/playlist?list=PLoO2tOP2r-XsXH0lybWl9zMvYp3lpLF46)

- [x] `Ableton Live For Beginners`

    - Midi track is for virtual instruments
    - Audio track is for outside sounds (samples) and instruments (guitar)
    - Operator synth
    - Arm a track and enable Computer Midi Keyboard to play notes
    - `z` and `x` to go up and down octaves. Drums need to have low octave selected to use keyboard
    - clips in session view.
    - copy and paste clips to other tracks (same melody but different instrument)
    - velocity to define how hard/soft a note is played
    - to record a clip with keyboard: 1. arm track, 2. play around, 3. set bpm, 4. hit Session Record button
    
- [x] `Ableton Live For Beginners (pt. 2)`

    - In a track enable `Auto` so you can hear what a microphone 
    - when recording something in preference-recording there is a count-in 
    - drag and drop a clip into arrangement view
