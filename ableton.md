# File Formats

- `.als` - Ableton Live Set - The main project file (arrangement, clips, devices, automation, etc.).
- `.alp` – Ableton Live Pack - Compressed package containing projects, samples, presets, etc.
- `.adg` – Audio Effect Rack
- `.adv` – Device Preset
- `.adg` – Instrument Rack
- `.agr`  – Groove file
- `.amxd` – Max for Live device
- `.aif` / `.aiff` – Audio sample
- `.wav` – Audio sample
- `.fla`c – Audio sample
